# Referência de Classes por Marco

Este diretório organiza as classes por marco arquitetural (M0, M1, M2), indicando o que foi criado ou modificado em cada etapa.

## Documentos

- `docs/classes/m0-classes.md`
  - Fundação de runtime (logging, tempo, input, ciclo de vida).
- `docs/classes/m1-classes.md`
  - Matemática, câmera, transform e renderer básico.
- `docs/classes/m2-classes.md`
  - Scene system, entidade/componentes, glTF loader base e asset manager.

## Convenção

Cada documento lista:

1. Classe/estrutura
2. Status no marco (`Criada` ou `Modificada`)
3. Arquivo
4. Métodos principais e pontos de atenção
